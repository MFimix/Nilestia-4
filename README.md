# Nilestia-4

> A unified Hyprland dotfile suite — **end-4 foundation** augmented with **Caelestia Shell** widgets.

![Nilestia-4 Banner](assets/preview.png)

## Architecture

```
end-4/dots-hyprland  (base: WM, bar, sidebar, overview, keybinds)
        │
        └── Nilestia-4 overlay (~/.config/quickshell/nilestia/)
               ├── Audio Manager Hub        (Win + A)
               ├── WiFi Hub                 (Win + Alt + W)
               ├── Bluetooth Hub            (Win + Alt + B)
               ├── Ethernet Hub             (Win + Alt + C)
               ├── Monitor Hub              (Win + Ctrl + M)
               ├── Top Menu (hot zone)      (hover top 2px)
               └── Caelestia Lockscreen     (Win + L)
```

## Quick Install

```bash
git clone https://github.com/MFimix/Nilestia-4
cd Nilestia-4
chmod +x install.sh
./install.sh
```

**Requires**: Arch Linux or derivative (CachyOS, EndeavourOS, Garuda, etc.)

## Keybindings

| Shortcut | Action |
|---|---|
| `Super + A` | Audio Manager |
| `Super + Alt + W` | WiFi Hub |
| `Super + Alt + B` | Bluetooth Hub |
| `Super + Alt + C` | Ethernet Hub |
| `Super + Ctrl + M` | Monitor Hub |
| `Super + L` | Lock Screen |
| Mouse → top 2px | Top Menu |
| Right-click WiFi icon (sidebar) | WiFi Hub |
| Right-click BT icon (sidebar) | Bluetooth Hub |
| *Dashboard button (top-left)* | Gemini AI panel |

> `Super + A` was **Gemini panel** in end-4. In Nilestia-4 the Gemini panel is accessed via the **top-left dashboard button** instead.

## Theming

Nilestia-4 uses **Material You Dark** palette generated by [matugen](https://github.com/InioX/matugen):

```bash
matugen image ~/Pictures/Wallpapers/your-wallpaper.jpg --mode dark
```

All Caelestia widgets inherit the generated palette via SCSS variables in  
`.config/quickshell/nilestia/styles/_nilestia-dark.scss`.

## Module Overview

### Audio Manager (`modules/audio/AudioManagerHub.qml`)
Full PipeWire audio control: output/input device selection, per-app volume sliders, mute toggles.

### Network Hubs (`modules/hubs/NetworkHubs.qml`)
- **WiFi**: Scan, connect, disconnect, WPA toggle
- **Bluetooth**: Pair, connect, disconnect, battery level, discovery scan
- **Ethernet**: Connection status, IP display, nm-connection-editor launcher

### Monitor Hub (`modules/monitor/MonitorHub.qml`)
Visual display layout preview, resolution/refresh/scale/rotation per monitor, `hyprctl` apply button.

### Top Menu (`modules/topmenu/TopMenu.qml`)
Caelestia-style dark top bar — appears on mouse hover at screen top (2px hot zone). Shows CPU/RAM stats, clock, app shortcuts, and quick-toggle buttons.

### Lockscreen (`modules/lock/NilestiaLock.qml`)
Caelestia lockscreen ported to `WlSessionLock`. Full systemd-logind integration via `nilestia-lock.service`.

## File Structure

├── install.sh                          # Main installer
├── README.md
├── .config/
│   ├── hypr/nilestia/
│   │   ├── nilestia.conf               # Master source file
│   │   ├── keybinds.conf               # Nilestia keybind overrides
│   │   ├── rules.conf                  # Window rules for all hubs
│   │   └── execs.conf                  # Startup with Nilestia QS profile
│   └── quickshell/nilestia/
│       ├── shell.qml                   # Entry point
│       ├── modules/
│       │   ├── audio/AudioManagerHub.qml
│       │   ├── hubs/NetworkHubs.qml
│       │   ├── lock/NilestiaLock.qml
│       │   ├── topmenu/TopMenu.qml
│       │   ├── monitor/MonitorHub.qml
│       │   └── sidebar/SidebarOverride.qml
│       └── styles/_nilestia-dark.scss

│   ├── nilestia-lock.sh                # Lockscreen activator

    └── icons/
